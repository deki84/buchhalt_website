# buchhalt.de – Bookkeeping Service Website

Professional website for a bookkeeping service in Augsburg.

🔗 **Live:** https://www.buchhalt.de/

## Features

- 🔍 SEO optimization with custom configuration
- 📊 Google Search Console integration
- 💼 Clean professional design for B2B clients
- 📱 Responsive design with Tailwind CSS
- ⚡ Fast loading with Next.js and Vercel deployment

## Tech Stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS
- **Deployment:** Vercel

## About

Real client project built and deployed for a bookkeeping service in Augsburg.
